Shell Commands Effect Examples
$ 'command' --help hows help for the shell command $ mv --help
$ clear Clears terminal screen
$ mv 'source' 'destination' Rename/move source to destination $ mv wronNameFile correctNameFile
$ rm Remove files and directories $ rmdir (Delete directories.)
$ rm -i valueable.txt (removes file with "interactive"/warning prompt)
$ mkdir and $ touch Commands to create new directory and files
$ start . Opens terminal directory folder in windows file explorer.
$ cd "path" Change working directory $ cd (change to top shell dierctory)
$ cd .. (change to parent dir. )
$ cd Documents/Yellowstone/Bears/ (Change Shell working directory to bear folder)
$ ls List the files in the directory. $ ls . = $ ls (list files in current dir.)
$ ls ~ (list files in my home dir.)
$ ls -l (option to list in a long /detailed form)
$ ls *.pdf (list of all .pdf files)
$ echo Print message back in terminal $ echo $COLUMNS x $LINES
; Allows 2 Shell commands on same line $ cd.. ; ls
pwd Print Working Directory $ pwd
$ curl "see url" is used in command lines or scripts to transfer data. $ curl 'http://google.com' (Show source code of a web page)
$ curl -L 'http:...' (Follow redirects)
$ curl -o google.html -L 'http://google.com' (save code directly to g.html file)
$ cat View/concatenate file content. $ cat text.txt (text content)
$ less View file content screen by screen. $ less longbigtext.txt (text content with space and arrows, "q" to quit)
$ wc "word count" reads either standard input or a list of files and generates one or more of the following statistics: newline count, word count, and byte count $c ws -l (line count)
$ grep "Global regular expression print,” processes text line by line and prints any lines which match a specified pattern. $ grep wordToFind textFile.txt (Output all lines containing wordToFind)
$ grep wordToFind textFile.txt | less (Output to Less which results in page by page find)
$ curl -L 'http...' | grep -c fish (-c counts how many fish in url)
variableName=Value Variables in Shell. More here $ varName='variable' (assignment)
$ echo $varName (Variable call)
$ echo $LINES x $COLUMNS (Internall Shell variables: prints terminal window dimension)
$ echo $PATH (Enviroment variable are shered within programs run from within the Shell: returns path to Program Shell files)
$ PS1 Controlling the shell prompt ($PS1) The tool is here by Julien Ricard. Try it out!
$ alias Allows you to create short names for commands. $ alias ll='ls -la'
$ alias (show all alias created)
